Arducurity - Proyecto de Seguridad con Arduino

Arducurity es un sistema de seguridad basado en Arduino que permite detectar intrusos mediante sensores de movimiento y alarma. Este proyecto se enfoca en la creación de dispositivos electrónicos conectados a la web.

Como parte de este proyecto, utilicé tecnología web para el control remoto del sistema, implementando una base de datos en MySQL para almacenar los registros de eventos y permitir el acceso remoto desde cualquier dispositivo.
